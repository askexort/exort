Free AI for Everyone + Token Tracker. Build autonomous agents with tool use, memory, and multi-provider support.

Features

🤖

Agentic Loop

Think → Act → Observe architecture for autonomous reasoning and decision making.

🔌

Multi-Provider

OpenAI, Ollama (local), Groq (free tier) out of the box. Switch providers with a single flag.

🛠️

Plugin Tools

Decorator-based tool registration system. Easy to extend with custom capabilities.

💾

Conversation Memory

SQLite-backed persistent conversations. Remember context across sessions.

⚡

Streaming

Real-time token streaming for all providers. See responses as they're generated.

📊

Token Tracker

Track token buy/sell events across multiple chains. Real-time alerts and analytics.

Quick Start

# Install Exort
pip install exort-agent

# Set up config
exort config --init

# Start chatting
exort chat

# Python API
from exort import Agent

agent = Agent(provider="groq")
response = agent.chat("What is the capital of France?")
print(response)
