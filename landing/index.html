Exort

THE OPEN AGENT ENGINE.

Exort is a self-hosted AI agent framework that gives you a fully autonomous assistant with tool use, persistent memory, and multi-provider support. Run it from your terminal, deploy it as a Telegram bot, or embed it anywhere. Your data stays yours. No vendor lock-in. No API fees required — use free providers like Groq and Cerebras, or bring your own keys.

Get Started Telegram Bot

⟩ How It Works

1

Install

One command to get started. Python package installs everything you need.

$ pip install exort-agent
2

Configure

Interactive setup walks you through provider and API key selection.

$ exort setup
# Choose provider & enter key
3

Chat

Drop into the interactive shell. Ask questions, run tools, test everything locally.

$ exort shell
exort> Hello!
exort> /tools
4

Deploy

Launch as a Telegram bot, Discord bot, or any interface. One command.

$ exort bot
# Bot is now live on Telegram

⟩ Features

Engine

The core reasoning loop. Handles multi-turn conversations, context management, and orchestrates tool calls autonomously. Supports streaming responses and graceful error recovery.

Core

GearBox

Extensible tool system. Built-in tools for code execution, web browsing, file management, and shell access. Write custom tools in Python — just decorate a function.

Tools

Memory

Persistent conversation memory that survives restarts. The agent builds understanding over time, remembering your preferences, past decisions, and ongoing projects.

Persistent

Providers

Nine LLM backends supported out of the box. Switch between Groq, OpenAI, Anthropic, Ollama, OpenRouter, Cerebras, SambaNova, HuggingFace, and Google AI Studio.

9 Backends

Playbooks

Inject domain knowledge into your agent. Markdown-based knowledge files that define expertise, behavior patterns, and specialized skills for any use case.

Knowledge

Telegram Bot

Full-featured Telegram deployment. Supports private chats, group conversations, inline queries, and command menus. Deploy with one command, share with anyone.

Interface
