How Exort Works

From install to production in four simple steps

1

Install

One command is all it takes. No cloning repos, no config files, no dependencies to wrangle. Works on Windows, Mac, and Linux.

pip install exort-agent
2

Configure

Interactive setup walks you through choosing an AI provider and adding your API key. Free providers like Groq work instantly with no credit card.

exort setup
3

Chat

Use the terminal shell for local development or deploy a Telegram bot for 24/7 access from anywhere. Send text, images, files, and voice messages.

exort shell / exort bot
4

Extend

Skills, memory, and tools grow over time. Create reusable procedures, build custom workflows, and watch your AI assistant become more capable with every conversation.

What Exort Can Do

Deep capabilities that make Exort the most versatile open-source AI agent

Tool Use & Execution

Exort doesn't just chat — it acts. Run Python code directly, execute shell commands, browse the web, manage files on your system, and make API calls. All through natural language. It can read and write files, install packages, run scripts, fetch web pages, search the internet, and chain multiple tools together to complete complex tasks autonomously.

Python Shell HTTP Files Search

Persistent Memory

Exort remembers. It automatically saves important facts from every conversation to a persistent memory store. It recalls past conversations, builds a growing understanding of your preferences and environment, and uses a skill system to save reusable procedures as markdown files. Session search lets you find anything from your history. Memory persists across restarts, so Exort is always learning.

Auto-Save Context Skills Sessions

Multi-Provider AI

Switch between free providers like Groq and Cerebras at zero cost, or use premium models from OpenAI and Anthropic when you need them. Run everything locally with Ollama for full privacy. Mix and match providers for different tasks. Auto-fallback ensures your agent never goes down. All configured through a single YAML file or the interactive setup wizard.

Frequently Asked Questions

Everything you need to know about Exort

Is Exort really free?

Yes. Exort is MIT licensed open source software. You can use free AI providers like Groq, Cerebras, and SambaNova to run AI at zero cost. You can also bring your own API keys for OpenAI, Anthropic, or any other provider. No subscriptions, no hidden fees.

Do I need a powerful computer?

No. Exort runs on any machine — even a Raspberry Pi. The AI processing happens on cloud providers (Groq, OpenAI, Anthropic), so your computer just sends and receives messages. If you want to run AI locally with Ollama, you'll need a decent GPU with 8GB+ VRAM.

How does the Telegram bot work?

Exort polls Telegram for new messages, processes them with AI, and replies. You need a bot token from @BotFather. Run exort bot and it works 24/7. It handles text, images, files, code execution, web browsing, and more. Deploy to a cloud server to keep it running when your PC is off.

How does memory work?

Exort automatically saves important facts from your conversations to persistent memory files. This memory is injected into every future conversation, so it remembers your preferences, environment, project details, and context. Memory persists across restarts and sessions. You can also create reusable "skills" — markdown files with step-by-step procedures.

What AI models does it support?

All major providers: Groq (Llama 3, Mixtral), OpenAI (GPT-4o, GPT-4), Anthropic (Claude), Ollama (any local model like Llama, Mistral, Phi), Cerebras, SambaNova, HuggingFace, Google AI Studio, and OpenRouter. You can switch providers anytime with a single command.

Can I use it for my business?

Yes. Exort is MIT licensed — use it for personal or commercial projects with no restrictions. Self-host for full control over your data. Build customer support bots, automate workflows, create internal tools, or integrate AI into your products.

Can I deploy it on a server?

Yes. Deploy to Render, Railway, Fly.io, or any VPS (DigitalOcean, Hetzner, AWS). The Telegram bot keeps running even when your PC is off. We have deployment guides for all major platforms. Render.com offers a free tier with no credit card required.

Is my data safe?

Yes. Exort runs locally on your machine. Your conversations, memory, and files stay on your computer. When using cloud AI providers, only the message text is sent to the API — your files and memory stay local. For maximum privacy, use Ollama to run AI entirely on your machine with zero data leaving your network.

Can Exort write and run code?

Yes. Exort has full tool use capabilities. It can write Python scripts, execute them in a sandboxed environment, manage files, browse the web, call APIs, run shell commands, search codebases, and much more. It runs code safely with timeouts and output limits.

How do I contribute?

Exort is open source on GitHub. Fork the repo, make your changes, and submit a pull request. You can contribute code, report bugs, suggest features, write documentation, or share skills. Join our Telegram group or Discord to connect with other contributors.
