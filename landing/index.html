Quick Start

Start Building in Minutes

Get up and running with Exort in just a few commands. No complex setup required.

  • Simple pip install
  • Free Groq API included
  • Python & CLI interface
  • Extensive documentation
View Documentation
Terminal Python 
# Install Exort
pip install exort-agent

# Set up config
exort config --init

# Start chatting
exort chat

# Use a specific provider
exort chat --provider groq
from exort import Agent

agent = Agent(provider='groq')
response = agent.chat('Hello!')
print(response)

# Streaming
for chunk in agent.chat_stream('Tell me a joke'):
    print(chunk, end='')

How It Works

Three Steps to Your AI Agent

1

Run pip install exort-agent to get the framework. Works on Windows, macOS, and Linux with Python 3.10+.

Build

FAQ

Frequently Asked Questions

Is Exort really free?

Yes. Exort is MIT licensed and 100% open source. You can use Groq's free tier for inference at zero cost. No hidden fees, no premium tiers.

Do I need a GPU?

No. Use Groq (free cloud) or OpenAI (cloud) and you don't need any local hardware. If you want to run models locally with Ollama, you'll need a decent GPU (8GB+ VRAM recommended).

What Python version do I need?

Python 3.10 or higher. We recommend using a virtual environment: python -m venv .venv && source .venv/bin/activate

How do I add custom tools?

Use the @tool decorator on any Python function. Exort automatically generates the JSON schema for the LLM. See the GitHub repo for examples.

Can I use it in production?

Exort is designed for production use with persistent memory, error handling, and streaming. However, as with any open-source project, test thoroughly before deploying.
