CLI Commands

Control Exort from your terminal

💬
Exort chat
Start an interactive chat session with an AI provider.
Exort chat [OPTIONS]
--providerLLM provider (openai, groq, ollama)
--modelModel name
--systemSystem prompt override
--max-tokensMax response tokens (default: 1024)
--temperatureTemperature (default: 0.7)
--streamEnable streaming (default: true)
⚙️
Exort config
Manage Exort configuration settings.
Exort config show # Show current config Exort config set KEY VALUE # Set a value Exort config path # Show config file path Exort config reset # Reset to defaults
🔌
Exort providers
List and test available LLM providers.
Exort providers # List all providers Exort providers test # Test connectivity
🔧
Exort tools
List and test available agent tools.
Exort tools # List all tools Exort tools test TOOL_NAME # Test a tool
🚀
Exort serve
Start a local API server (coming soon).
Exort serve --port 8080 --host 0.0.0.0

Python API

Build programmatically with Exort

🤖
Agent
Create an agent and chat with AI, with or without tools.
from Exort import Agent, Config config = Config() agent = Agent(config) # Simple chat response = await agent.chat("What is 2+2?") # Chat with tool use response = await agent.chat("Search the web for AI news") # Streaming async for chunk in agent.stream("Tell me a story"): print(chunk, end="")
⚙️
Config
Load and manage configuration from files or environment.
from Exort import Config config = Config() config.get("provider") # "groq" config.set("model", "llama-3.3-70b") # Load from file config = Config.from_file("~/.Exort/config.yaml") # Override with env vars config = Config.from_env()
🔌
Providers
Use providers directly for custom integrations.
from Exort.providers.groq import GroqProvider provider = GroqProvider(api_key="gsk_...") # Non-streaming response = await provider.chat( messages=[{"role": "user", "content": "Hello"}], model="llama-3.3-70b-versatile" ) print(response.content) # Streaming async for chunk in provider.stream(messages=[...]): print(chunk, end="")

Tools & Memory

Extend Exort with custom tools and persistent memory

🔧
Custom Tools
Register custom tools with the decorator pattern.
from Exort.tools.base import tool, ToolRegistry @tool( name="my_tool", description="Does something cool", parameters={ "input": {"type": "string"} } ) async def my_tool(input: str) -> str: return f"Processed: {input}" registry = ToolRegistry() registry.discover() result = await registry.execute( "web_search", query="hello" )
🧠
Memory Store
Persistent memory for multi-turn conversations.
from Exort.memory.store import MemoryStore store = MemoryStore() # Save messages store.save_message( user_id="user1", role="user", content="Hello" ) # Get history history = store.get_history( user_id="user1", limit=10 ) # Stats stats = store.get_stats(user_id="user1")
