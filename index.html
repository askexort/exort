Free AI for Everyone. An autonomous AI agent with tool use, memory, and multi-provider support. Chat from Telegram, CLI, or any interface you already use.

⟩ What It Does

Runs on Your Machine

Full control and privacy. Your data never leaves your infrastructure. Self-host with zero dependencies.

Any Chat App

Connect via Telegram, CLI, Discord, Slack, or any interface you already use. One agent, every platform.

Persistent Memory

Remembers context across conversations. Builds understanding over time for smarter, more relevant responses.

Tool Use

Executes code, browses the web, manages files, and calls APIs. An agent that actually does things.

Multi-Provider

Switch between Groq, OpenAI, Anthropic, Ollama, and more. Use free providers or your own API keys.

Open Source

MIT licensed and fully transparent. Inspect, modify, and contribute. Community-driven development.
