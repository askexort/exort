Quick Start

Start Building in Minutes

Get up and running with Exort in just a few commands. No complex setup required.

  • Simple pip install
  • Free Groq API included
  • Python & CLI interface
  • Extensive documentation
View Documentation
Terminal Python 
# Install Exort
pip install exort-agent

# Set up config
exort config --init

# Start chatting with Groq (free)
exort chat --provider groq

# Use OpenAI
exort chat --provider openai --model gpt-4o
# Import Exort
from exort import Agent

# Create an agent
agent = Agent(provider="groq")

# Chat with the agent
response = agent.chat("Hello!")
print(response)

# Stream responses
for chunk in agent.chat_stream("Tell me a joke"):
    print(chunk, end="")

# Check stats
print(agent.get_stats())

FAQ

Frequently Asked Questions

Is Exort really free?

Yes! Exort is open source under the MIT license. You can use it for personal and commercial projects. You only pay for the AI provider API usage (Groq offers a free tier).

Do I need an API key?

For cloud providers like Groq or OpenAI, yes. But if you use Ollama for local models, no API key is needed — everything runs on your machine.

Can I use it with my own tools?

Absolutely! Exort has a plugin-based tool system. Create custom tools by defining a function and registering it with agent.add_tool().

Which model should I start with?

For beginners, we recommend Groq with llama-3.3-70b-versatile — it's free and fast. For production, use OpenAI gpt-4o for best results.

How does memory work?

Exort supports short-term (conversation) and long-term (persistent) memory. Agents remember context across turns and can store important information for future reference.
