Quick Start

Start Building in Minutes

Get up and running with Exort in just a few commands. No complex setup required.

  • Simple pip install
  • Free Groq API included
  • Python & CLI interface
  • Extensive documentation
View Documentation
Terminal Python 
# Install Exort
pip install exort-agent

# Set up config
exort config --init

# Start chatting
exort chat

# Python API
from exort import Agent

agent = Agent(provider="groq")
response = agent.chat("Hello!")
print(response)
