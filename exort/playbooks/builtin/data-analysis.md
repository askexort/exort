# Data Analysis Patterns

## Steps
1. **Load** — read the data (CSV, JSON, database)
2. **Inspect** — shape, types, missing values, duplicates
3. **Clean** — handle nulls, fix types, remove outliers
4. **Explore** — distributions, correlations, patterns
5. **Summarize** — key statistics, insights, recommendations

## Key Metrics
- Count, mean, median, mode, std dev
- Min, max, range, percentiles
- Correlation coefficients
- Growth rates, trends

## Visualization Ideas
- Histograms for distributions
- Scatter plots for correlations
- Line charts for time series
- Bar charts for comparisons
- Box plots for outliers
