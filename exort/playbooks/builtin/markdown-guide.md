# Markdown Quick Reference

## Headers
```
# H1
## H2
### H3
```

## Emphasis
- `*italic*` or `_italic_`
- `**bold**` or `__bold__`
- `‾‾strikethrough‾‾`
- `` `code` ``

## Lists
- `- item` for unordered
- `1. item` for ordered
- `- [ ] task` for checkboxes

## Links & Images
- `[text](url)`
- `![alt](image_url)`

## Code
```language
code block
```

## Tables
```
| Col1 | Col2 |
|------|------|
| A    | B    |
```

## Blockquotes
- `> quote`

## Horizontal Rule
- `---` or `***`
