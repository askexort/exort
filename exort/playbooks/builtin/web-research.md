# Web Research Workflow

When researching a topic:

1. **Define the question** — what exactly do you need to find?
2. **Search broadly first** — use `web_search` with 2-3 different queries
3. **Read top results** — use `fetch_url` on the most relevant URLs
4. **Cross-reference** — verify facts across multiple sources
5. **Synthesize** — combine findings into a clear answer with citations

## Tips
- Use specific queries over vague ones
- Try different phrasings if first search is weak
- Always cite sources with URLs
- Distinguish facts from opinions
- Check dates — prefer recent information
