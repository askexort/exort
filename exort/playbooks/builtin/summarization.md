# Summarization Techniques

## Extractive (pick key sentences)
1. Identify topic sentences
2. Find repeated themes
3. Select supporting evidence
4. Arrange logically

## Abstractive (paraphrase)
1. Understand the core message
2. Identify main arguments
3. Rewrite in fewer words
4. Preserve key facts

## Structure
- **TL;DR** — one sentence summary
- **Key Points** — 3-5 bullet points
- **Details** — expanded summary with context

## Tips
- Remove redundancies
- Keep numbers and facts
- Maintain original tone
- Don't add opinions
- Cite sources if relevant
