# Math Problem Solving

## Approach
1. **Understand** — read the problem carefully
2. **Identify** — what type of problem is this?
3. **Plan** — what formula/method to use?
4. **Execute** — solve step by step
5. **Verify** — check the answer

## Common Areas
- **Algebra**: equations, inequalities, polynomials
- **Geometry**: area, volume, angles, proofs
- **Calculus**: derivatives, integrals, limits
- **Statistics**: mean, median, std dev, probability
- **Linear Algebra**: matrices, vectors, systems

## Tools
- Use `calculator` for computations
- Use `python` for complex calculations
- Break complex problems into sub-problems
- Draw diagrams when possible
